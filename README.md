## 東京都北区の保育園 パパ・ママ交流スペース
https://rails-app-portfolio-11fcb35a8b51.herokuapp.com/

北区の保育園に在園中の保護者や保活中の方と気軽に投稿・コメントを通じて交流できるアプリケーションです。  
現在地から近くの保育園を探したり、エリアで絞ったり、フリーワードで検索して気になる保育園を探せます。  
ログインすると、保育園の内情を聞いたり情報交換をしたり出来ます。  
  
### 開発の経緯
私自身2児の母で保活を経験してきましたが、  
特に第一子の時はママ友もおらず、実際に子供を連れて足を運んで情報を得るには労力がかかりすぎてしまったり、  
ネット上にも保育園の実際の口コミなどが少なかったり古い情報しか無くて困りました。  
また、北区は新園の設立も多く、最新の園の情報はさらに入手しづらかったのです。  
そこで、保護者同士が気軽に質問したり相談できる場所が欲しいと思い、このアプリケーションを作成しました。  
入園した後も、仕事の時間がバラバラで中々会えない保護者が多かったり、ゆっくり話す時間もなかったため  
在園児の保護者同士も相談したり交流できる場になればいいなと思っています。

### 想定ユーザー
- 保育園に通っている子どもの保護者
- 保育園を探している保護者
  
### 解決したい課題
- いろんな園の情報や口コミを知りたいが、子供を連れて中々見学に行けない  
- 地域にママ友などのコミュニティがなく情報がわからない  
- 保育園の送迎時間が重ならず、普段会えない保護者と話をしたい

### 機能一覧
| 機能 | 説明 |
| ---- | ---- |
| ユーザー登録 | ユーザーの新規登録・編集・削除ができます(CRUD処理使用) |
| ログイン機能 | 名前、メールアドレスの入力でログイン・ログアウトができます(sessionコントローラー使用) |
| ゲストログイン | 新規登録せずにログイン時と同じ状態を試せます |
| 保育園検索(フリーワード検索) | 保育園名をフリーワードで検索できます |
| 保育園検索(現在地検索) | geocorderで現在地を取得し、ピンで表示された近い保育園をクリックすると詳細ページへ遷移します(google maps API使用) |
| 保育園検索(エリア別検索) | 気になるエリア名のボタンをクリックするとそのエリア名を含む住所の保育園が検索できます |
| 保育園お気に入り登録 | 気になる保育園にハートをつけるとお気に入り登録され、マイページで一覧表示されます(JavaScript使用) |
| マイページ | 名前、アイコン、自己紹介、お気に入り保育園一覧、自分の投稿一覧が表示されます |
| 投稿機能 | それぞれの保育園に質問や相談などを投稿でき、自身の投稿のみ編集と削除可能です(CRUD処理使用) |
| コメント機能 | 投稿に対してコメントができます/自身のコメントのみ削除可能です |
| 吹き出しアイコン | 投稿・コメントのユーザー名右横の吹き出しアイコンにカーソルを合わせると各ユーザーの自己紹介文が表示されます |
| レスポンシブ対応 | 画面サイズに合わせて表示されます(Bootstrap使用) |
  
### 工夫した点
- パッと見て何のボタンかわかるシンプルなデザインと機能にし、忙しい保護者が感覚的に使いやすいようにしました
- トップページで3つの検索軸を作り、保育園を探しやすくしました
- 北区在住の方をメインターゲットとするため、現在地を取得し近隣の保育園を表示し、ピンをクリックするとその保育園のページに飛べるようにしました
- 気になる保育園にハートをつけるとお気に入り登録され、マイページでお気に入り保育園が一覧で確認できるようにしました
